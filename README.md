# Neovim Config

### Clean Neovim

```
rm -rf ~/.cache/nvim
rm -rf ~/.local/share/nvim/
rm -rf ~/.local/state/nvim
```

### Instructions

**Config (Mac & Linux)**

```bash
git clone git@github.com:chrsolr/neovim-config.git ~/.config/nvim
```

**Config (Windows)**

```bash
git clone git@github.com:chrsolr/neovim-config.git %userprofile%\AppData\Local\nvim\
```
